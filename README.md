# Milokids English

Milokids is an educational app designed for children, helping them learn English through interactive activities and games. The app includes various modes such as audio learning, storytelling, and creative activities.

## Features
- Splash screen showcasing the app's purpose.
- Main menu allowing users to select categories.
- Category screens featuring different themes.
- Multiple activity modes to enhance learning experience.

## Installation
To install the Milokids English app, clone the repository and run the following command:
```bash
flutter pub get
```

## Getting Started
To launch the app, run:
```bash
flutter run
```
